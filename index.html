Task Time Tracking App

The Task Time Tracking app is an iOS app that facilitates a job costing system and serves as a working task list for the user. The app enhances and optimizes the task's particular workflow from beginning to end. Starting with capturing task time, to the second, and finishing with multiple workflows depending on if the task was completed successfully. All guided by the app. The time captured on the app feeds back into a job costing system that enhances task and employee scheduling in the future. It also provides performance metrics about the user doing the task. This information feeds into a quality systems for training and evaluations processes.

The system was built using:

Swift
iOS
Node.js
JavaScript

Task Assignment App

The Task Assignment app is an iOS app that facilitates the supervisor in assigning a task to their employees. It is an intuitive interface that is paired with a bluetooth handheld barcode scanner that allows the supervisor to assign tasks in a quick down-the-list type style. With each task assignment the app factors in work time requirements for each task, allowing the supervisor to see in real time the work load assigned to each employee. The mobile nature of the device allows the supervisor to quickly move between work ares and openly discus the assignments with their employees. Used by working supervisors, the app has optimized their assignment responsibilities freeing up their time to focus on their job functions.

The system was built using:

Swift
iOS
Node.js
JavaScript

Rotable Exchange Inventory Management System

The Rotable Exchange Inventory Management System is an elegant solution to a complex inventory process of exchanging items in and out of warehouse to expedite the product to the customer. I re-engineered this process based on a legacy ERP system that had average processing time of 25 minutes per item, with an average of 90 clicks across 6 different screens. My solution reduced the processing time to an average of 6-minute processing time, with an average of 12 clicks across two screens. This system also included much needed features that were not available in the legacy ERP, such as a batch processing, alerts, quality required hard stops, automated warehouse and shipping document creation, and robust warehouse activity reporting tools. This system was recently approved for a $1+ million investment and is quickly becoming a core work flow for the company.

Yamazumi Board: Work Assignment Tool

The Yamazumi board is an interactive work assignment tool. It visually shows what work is currently not assigned (left two columns) and the work currently assigned to each team member. The work assignment is done by bragging and dropping a work square from on column to another. As a work square is assigned to a team member, their available work time (seen in blue below the team member name) decreases based on the work blocks time to complete. Subsequently, if a work block is pulled away from a team member, their available work time increases. This tool gives the team lead visual way to understand the current work demands on each employee, while at the same time allowing a team member to understand their specific demands. This tool was designed to be displayed on a large screen in the work area to allow a for a high level of transparency about what each team member is assigned, and how much work each team member is assigned.

The system was built using:

HTML/CSS
jQuery
JavaScript
jQuery UI
ASP (Legacy Backend)
